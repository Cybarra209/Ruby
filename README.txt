Ruby-the-Battle

How to play:
- Pick Red, Blue, or Yellow ship.
- Hold left or right to move.
- Hold fire to shoot.
- You can press fire while moving because the controls support multitouch.
- Defeat Ruby before your 5 health bars run out.

GitHub Pages Setup:
1. Download and unzip RubyTheBattle_FULL.zip.
2. Go to https://github.com and sign in as Cybarra209.
3. Click New Repository.
4. Name it: Ruby-the-battle
5. Make it Public.
6. Click Create Repository.
7. Click Add file > Upload files.
8. Drag in index.html, style.css, game.js, README.txt, and the assets folder.
9. Click Commit changes.
10. Go to Settings > Pages.
11. Under Build and deployment, set Source to Deploy from a branch.
12. Branch: main
13. Folder: / (root)
14. Click Save.
15. Wait 1 to 2 minutes.
16. Your game should be live at:
    https://cybarra209.github.io/Ruby-the-battle/

Important:
GitHub Pages is case-sensitive. Keep the assets folder and file names exactly as they are.
